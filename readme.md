# metre

Install:

- https://git-scm.com/
- https://www.meteor.com
- https://nodejs.org

Then:

```
git clone https://github.com/Snapper1995/metre.git cherenkovatel
cd cherenkovatel
meteor npm i
meteor npm start
```

Launch parallel two processes. CNCJS on 8000 port (http://localhost:8000) and current app on 3000 port (http://localhost:3000).
